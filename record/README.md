# record 文件夹

该文件夹包含记录管理相关的实现，负责数据库表的记录存储和访问。

## 内容

- **Layout.c/h**: 记录布局管理，定义表中记录的格式和结构。
- **RID.c/h**: 记录标识符，用于唯一标识表中的记录。
- **RecordPage.c/h**: 记录页管理，负责在数据页中存储和访问记录。
- **Schema.c/h**: 表模式管理，定义表的字段和数据类型。
- **TableScan.c/h**: 表扫描，提供对表中记录的顺序访问。

## 用途

- 管理表中记录的存储格式
- 提供记录的插入、删除、更新和查询操作
- 处理记录的物理存储和逻辑表示之间的映射
- 支持记录的顺序访问和随机访问
- 管理记录的标识符和定位
