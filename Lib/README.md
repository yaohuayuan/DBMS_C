# Lib 文件夹

该文件夹包含数据库系统的基础库实现，提供各种数据结构和工具函数。

## 内容

- **BlockLockManager.c/h**: 块锁管理器，用于管理数据块的锁定机制。
- **ByteBuffer.c/h**: 字节缓冲区，用于高效处理字节数据。
- **CCMath.h**: 数学工具函数。
- **CMap.c/h**: 映射数据结构实现。
- **CString.c/h**: 字符串处理工具。
- **CVector.c/h**: 向量数据结构实现。
- **Error.c/h**: 错误处理机制。
- **List.c/h**: 链表数据结构实现。
- **RBT.c/h**: 红黑树数据结构实现。
- **StreamTokenizer.c/h**: 流分词器，用于解析输入流。
- **Trie.c/h**: 字典树数据结构实现。
- **map.c/h**: 映射数据结构的另一种实现。
- **rwlock.h**: 读写锁实现。

## 用途

- 提供各种基础数据结构
- 实现通用工具函数
- 支持数据库系统的各个模块
- 提供内存管理和错误处理机制
