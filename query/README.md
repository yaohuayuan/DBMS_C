# query 文件夹

该文件夹包含查询执行相关的实现，负责执行生成的查询计划并返回结果。

## 内容

- **Constant.c/h**: 常量值的表示和处理。
- **Expression.c/h**: 表达式的表示和求值。
- **Predicate.c/h**: 谓词的表示和求值，用于WHERE子句。
- **ProductScan.c/h**: 乘积扫描，实现表连接操作。
- **ProjectScan.c/h**: 投影扫描，实现SELECT子句的投影操作。
- **Scan.c/h**: 扫描操作的基类和通用接口。
- **SelectScan.c/h**: 选择扫描，实现WHERE子句的选择操作。
- **Term.c/h**: 谓词中的条件项，由字段、运算符和值组成。
- **UpdateScan.c/h**: 更新扫描，支持对数据的修改操作。

## 用途

- 执行查询计划中的各种操作
- 实现数据的扫描、过滤、投影和连接
- 处理查询中的表达式和谓词
- 支持数据的查询和更新操作
- 提供查询结果的访问接口
