# buffer 文件夹

该文件夹包含缓冲区管理相关的实现，负责数据库系统的内存缓冲区管理和页面替换策略。

## 内容

- **LRU/**: 包含LRU（最近最少使用）页面替换算法的实现
  - **LRUCore.c/h**: LRU核心算法实现
  - **LRUPolicy.c/h**: LRU策略接口实现
- **Buffer.c/h**: 缓冲区实现，管理内存中的数据页
- **BufferManager.c/h**: 缓冲区管理器，负责缓冲区的分配、回收和替换
- **ReplacementPolicy.h**: 页面替换策略的抽象接口

## 用途

- 管理数据库系统的内存缓冲区
- 减少磁盘I/O操作，提高系统性能
- 实现各种页面替换算法（如LRU）
- 支持并发事务的缓冲区访问
- 提供缓冲区与磁盘之间的数据同步机制
