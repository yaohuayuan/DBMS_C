# tx 文件夹

该文件夹包含事务管理相关的实现，负责数据库系统的事务处理和并发控制。

## 内容

- **concurrency/**: 并发控制相关实现
  - **ConcurrencyManager.c/h**: 并发管理器，负责事务的并发控制。
  - **DeadlockDetector.c/h**: 死锁检测器，用于检测和处理死锁。
  - **LockTable.c/h**: 锁表，用于管理事务的锁请求和分配。
- **recovery/**: 事务恢复相关实现
  - **LogRecord.c/h**: 日志记录，包含事务操作的日志条目。
  - **RecoveryManager.c/h**: 恢复管理器，负责事务的恢复操作。
- **BufferList.c/h**: 缓冲区列表，管理事务使用的缓冲区页。
- **Transaction.c/h**: 事务实现，代表一个数据库事务。
- **TransactionManager.c/h**: 事务管理器，负责事务的创建、提交和回滚。

## 用途

- 实现事务的ACID特性（原子性、一致性、隔离性、持久性）
- 管理事务的并发执行，防止数据不一致
- 处理事务的提交和回滚
- 实现锁机制，解决并发冲突
- 检测和处理死锁情况
- 实现事务恢复机制，确保系统崩溃后的一致性
