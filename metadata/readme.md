# metadata 文件夹

该文件夹包含元数据管理相关的实现，负责数据库系统的元数据存储和管理。

## 内容

- **IndexInfo.c/h**: 索引信息管理，存储和检索索引的元数据。
- **IndexManager.c/h**: 索引管理器，负责索引的创建、删除和维护。
- **MetadataManager.c/h**: 元数据管理器，协调各种元数据的管理。
- **StatInfo.c/h**: 统计信息管理，存储和检索表的统计数据。
- **StatManager.c/h**: 统计管理器，负责收集和维护表的统计信息。
- **TableManager.c/h**: 表管理器，负责表的创建、删除和维护。
- **ViewManager.c/h**: 视图管理器，负责视图的创建、删除和维护。

## 用途

- 管理数据库系统的元数据
- 存储表、索引、视图等对象的定义信息
- 支持元数据的查询和修改
- 维护表的统计信息，用于查询优化
- 协调各种元数据对象的管理
